Welcome to our Prometeo 2023 HackoverFlow Project, which comprises of a resume building website.
Creating a resume is a bit tedious task for any working professional from any industry. One has to keep it short, simple, with the latest work experience and constantly update it over a while. It is an application that simplifies the task of creating a resume for individuals. The system is flexible to be used and reduces the need of thinking and designing an appropriate resume according to qualifications. The system is developed to provide an easy means for creating a professional looking resume.
